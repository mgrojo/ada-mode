AUnit_ext (Stephe's AUnit extensions)

Current version: 1.0

AUnit_ext provides generic Check procedures, to allow writing AUnit tests like this: 

    Check (Message => "square 2", Computed => 2**2, Expected => 4);
There are generics for access, discrete, integer, fixed point types, arrays (1D, 2D, constrained/unconstrained, some with tolerance), etc.

AUnit_ext is distributed in source form only: org.stephe_leake.aunit_ext-1.0.tar.bz2
