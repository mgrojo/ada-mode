OpenToken Developer tools

Running the OpenToken tests requires AUnit. Compiling with the '-gnatyO' switch requires patching the compiler run-time sources in GNAT 6.2.1.

AUnit

The version of AUnit in Debian 5.0 is 1.03. The version distributed by AdaCore with GNAT 6.2.1 and GNAT GPL-2009 is 3.1.1. The latest AdaCore AUnit, distributed with GNAT 7.0.2, is AUnit 3.4.2. They have different user interfaces; the later versions are compatible with the Ravenscar profile.

The tests in OpenToken use AUnit 4.3.2. It can be built and used in place; it does not have to be installed in the GNAT installation directory.

  1. Download aunit-3.4.2-src.tgz from AdaCore GNAT Pro, or aunit-gpl-2012-src.tgz from AdaCore GNAT Libre.
  2. Untar the source anywhere; here we assume /Projects/aunit-3.4.2-src, then compile it: 
    cd /Projects/
tar zxf aunit-gpl-2012-src.tgz
cd aunit-3.4.2-src
make
  3. Edit opentoken/build/*/Makefile; set ADA_PROJECT_PATH to include /Projects/aunit-3.4.2-src/lib/gnat/
