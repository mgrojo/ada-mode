OpenToken Developer tools

OpenToken has been tested with GNAT GPL 2014 on Windows and Debian. Here we give installation instructions for these tools on Windows and Debian.

As of the OpenToken release date, the Debian stable GNAT package does not support Ada 2012 containers, so the GNAT GPL 2014 package from AdaCore must be used.

On Windows, Cygwin is used to run the Makefiles.

GNAT GPL 2014

Download and run GNAT GPL 2014 from AdaCore GNAT Libre.

On Linux, I recommend installing to /opt/gnat/gpl-2012/. In addition, you will need to add: 

      export LIBRARY_PATH=/usr/lib/i386-linux-gnu/

Running the tests requires additional packages on Cygwin and Debian:

  • dos2unix
